What is PIAGETS?

PIAGETS — Parameter-Importance-weighted Attractor-Graph Embedding for Task Schematization — is a continual learning framework for dynamical systems reconstruction (DSR). Given a stream of time-series from different chaotic or periodic attractors, the model must reconstruct each one without catastrophically forgetting prior tasks.

Assimilation vs Accommodation

  • Assimilation: new data fits an existing schema — reinforce with high $\lambda$
  • Accommodation: new data doesn’t fit — build a new schema with low $\lambda$

The schema probe compares $\phi(\theta)$ to known class centroids before each task and selects $\lambda$ automatically.

Core Challenge

Standard EWC uses the prediction Fisher $F_i^{\text{pred}} = \mathbb{E}[(\partial \hat{x}_t / \partial \theta_i)^2]$, which explodes for chaotic systems: gradients grow as $e^{\lambda_{\max} t}$ along trajectories.

PIAGETS replaces it with the Signature-Fisher:

$$F_i^\Phi = \left\|\frac{\partial\,\phi(\theta)}{\partial\,\theta_i}\right\|^2$$

$\phi$ is a stable ergodic attractor signature, so $\nabla_\theta \phi$ is bounded regardless of $\lambda_{\max}$.

Pipeline

  1. Train AL-RNN on task $t$ with GTF-BPTT and EWC penalty weighted by $F_i^\Phi$ from task $t-1$
  2. Compute $\phi(\theta_t) \in \mathbb{R}^{24}$ — differentiable attractor signature
  3. SLAO: QR-orthogonalise $W_A$; Fisher-weighted EMA of $W_B$ (merge before forget)
  4. Schema probe: $d_k = \|\phi - \bar\phi_k\|$ → choose $\lambda$ for next task
